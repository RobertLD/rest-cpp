Detailed Description

template<typename T>
class rest_cpp::Result< T >

Result<T> represents either a successful value of type T or an Error.

Template Parameters
TThe type of the successful value.
Note
This is similar to std::expected<T, Error> in C++23. The goals here are to provide a flexible and type correct way of interacting with a HTTP Result.

Return the stored value if present, otherwise call a fallback function.

This is lazy fallback:

  • make_fallback() is only invoked when there is no value.
  • avoids constructing expensive defaults unnecessarily.
    Template Parameters
